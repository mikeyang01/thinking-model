# 批判性思维

## 简介
批判性思维强调基于证据与逻辑，对信息进行分析、质疑、验证与评估，避免认知偏差与草率结论。

## 用法
- 明确主张与论据的关系，辨别因果与相关。
- 检查信息来源的可靠性与样本代表性。
- 列出反例与反驳，进行对立观点检验。

## 例子
市场数据评估清单（可复用）

1) 采样：样本量、覆盖城市、时间窗口是否与目标匹配；
2) 指标：定义是否一致（DAU/MAU口径、去重规则）；
3) 因果：是否存在混杂变量（节假日、补贴）；
4) 相关：展示相关系数与置信区间；
5) 反例：选择3个极端样本进行反驳检验；
6) 复现：独立来源复核一次；

应用结果：一次投放评估中剔除了节假日导致的虚高样本，ROI从 2.1 修正为 1.6。
