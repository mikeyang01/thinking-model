# SWOT分析模型

## 简介
用于评估内部优势/劣势与外部机会/威胁，辅助战略定位与方案选择。

## 用法
- 收集内外部信息，填充四象限。
- 生成SO/WO/ST/WT策略组合。
- 结合数据指标进行优先级排序。

## 例子
进入城市X的SWOT与策略组合

- S：本地配送网络覆盖 85%，履约成本较低；
- W：品牌认知度低，地推团队不足；
- O：高校聚集，外卖需求上升；
- T：头部竞争者强势补贴；
- 策略：
  - SO：用低履约成本+校园渠道做套餐定价；
  - WO：与社团合作提升认知，短期外包地推；
  - ST：以差异化品类避开正面补贴战；
  - WT：控制补贴上限，设置止损阈值；
- 指标：三月内市占≥8%，CAC≤60元；未达则缩减投放。
