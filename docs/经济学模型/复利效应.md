# 复利效应

## 简介
收益再投入会产生“利滚利”，长期小幅优势可累计成巨大差异。

## 用法
- 尽早开始、持续稳定、降低波动。
- 将学习与产品指标也纳入“复利化”设计。
- 避免大幅回撤破坏复利轨迹。

## 例子
定投与回撤控制

- 计划：每月固定投入 2000 元，回撤>15%时暂停加仓，恢复后分三月补回；
- 五年后IRR 对比：有回撤控制的策略 IRR 高 1.8pp；
- 解释：降低大幅回撤有助于长期复利。

产品体验的复利

- 每周发布 1-2 个小改进，设“复利看板”：满意度、留存、关键动作转化；
- 半年内满意度 4.2→4.6，次月留存 +5pp；
- 小幅持续叠加带来长期显著提升。
